# 💼 Matheus | Frontend Developer Portfolio

A modern and minimal portfolio focused on **UI/UX Design and Frontend Development**, showcasing projects, skills and a clear development process.

---

## 🚀 About the Project

This portfolio was designed and built to present my work in a **clean, professional and impactful way**.

It combines:

* Thoughtful UI design
* Clean and maintainable code
* Smooth animations and interactions
* Fully responsive layout

---

## 🎯 Features

* ⚡ Modern and minimal UI
* 📱 Fully responsive (mobile, tablet and desktop)
* 🎨 Custom design system (colors, typography, spacing)
* 🧠 “My Process” section with animated timeline
* ✨ Scroll-based animations (Intersection Observer)
* 🔥 Interactive elements with hover effects
* 🌍 Ready for international clients (EN version)

---

## 🛠️ Tech Stack

* HTML5
* CSS3 (Flexbox + Grid)
* JavaScript (Vanilla JS)
* Intersection Observer API
* Font Awesome (icons)

---

## 📌 Sections

* **Hero** → Introduction and positioning
* **About Me** → Background and approach
* **Skills** → Technologies and tools
* **Projects** → Selected work
* **Process** → From idea to delivery (animated timeline)
* **Contact** → Call to action

---

## 🧠 What I focused on

This project emphasizes:

* Clean layout structure
* Visual hierarchy
* User experience
* Performance and simplicity
* Real-world portfolio standards

---

## 📱 Responsiveness

The layout adapts to:

* 📱 Mobile devices
* 📲 Tablets
* 💻 Desktop screens

Using a mobile-first approach with optimized breakpoints.

---

## ✨ Animations

* Scroll-triggered animations using **Intersection Observer**
* Sequential reveal of process steps
* Animated timeline (growing line)
* Smooth hover interactions
